# Sorting Visualizer

An interactive web application that helps users learn and understand sorting algorithms through real-time visualizations, algorithm explanations, complexity analysis, and code examples.

## Live Demo

https://sorting-visualizer-eta-five.vercel.app/

## Features

* Visualize sorting algorithms with animations
* Generate and sort custom arrays
* Adjust array size and sorting speed
* Track comparisons, swaps, and execution time
* View time and space complexity of algorithms
* Code examples in Python, Java, and C++
* Practice problems for algorithm learning
* Responsive and interactive UI

## Algorithms Implemented

* Bubble Sort
* Insertion Sort
* Selection Sort
* Merge Sort
* Quick Sort
* Heap Sort
* Counting Sort
* Radix Sort

## Tech Stack

* React.js
* JavaScript
* HTML5
* CSS3
* Vercel

## Getting Started

Clone the repository:

```bash
git clone https://github.com/GauravS28/sorting-visualizer.git
cd sorting-visualizer
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
